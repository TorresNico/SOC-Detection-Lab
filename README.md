# SOC-Detection-Lab

## Overview
This project demonstrates hands-on SOC analyst skills using Splunk, Windows Security Logs, Sysmon, and Kali Linux in a virtualized lab environment.

The lab focuses on:
- Detection engineering
- Security event analysis
- Simulated attacker behavior
- Incident investigation workflows

---

## Technologies Used
- Splunk Enterprise
- Windows Event Logs
- Sysmon
- Kali Linux
- VirtualBox
- PowerShell

---

## Detections Developed

### 1. Authentication Abuse Detection
- Event ID: 4625
- Identified repeated failed login attempts
- Simulated brute-force behavior

### 2. Suspicious Process Execution
- Event ID: 4688
- Monitored PowerShell and command-line activity
- Investigated suspicious execution behavior

### 3. Network Reconnaissance Detection
- Simulated reconnaissance activity using Kali Linux
- Analyzed command execution activity associated with attacker behavior
